# git remote


해당 저장소에 있는 리모트 저장소에 대한 정보를 알기 위해서는 아래 명령어를 사용하시면 됩니다
참고로 원격 저장소를 처음 clone 했을 경우 기본으로 해당 주소가 origin에 들어가 있으니 참고하시기 바랍니다.
​
$ git remote -v
origin  https://github.com/shinjihyun/LearnToPython.git (fetch)
origin  https://github.com/shinjihyun/LearnToPython.git (push)


* git remote rm [remote repository ] 으로 삭제하고 새롭게 명명을 할 수 도 있습니다.
