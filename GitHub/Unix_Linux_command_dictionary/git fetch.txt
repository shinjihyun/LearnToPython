[git fetch]

+ 사용법
- git fetch 원격_저장소_주소 [원격저장소_브랜치명:로컬저장소_브랜치명]
- git fetch remote-name

+설명 및 예제
fetch는 원격 저장소로부터 소스를 가져오는 명령어.
원격 저장소의 project_server이라는 브랜치를 로컬 저장소로 가져와 project_local이라는 브랜치를 생성
$ git fetch git@github.com:pirania/hanbit2.git project_servwe:project_local

현재 시점으로 원격 저장소에서 최종 변경된 소스를 가져와보자.
가져온 소스는 FETCH_HEAD라는 이름의 브랜치로 자동 생성.
$git getch git://android.git.kernel.org/device/common.git common master 
from git://android.git.kernel.org/device/common.git
* branch		master -> FETCH_HEAD
$ git diff FETCH_HEAD

연결된 원격 서버의 브랜치를 모두 가져오는 명령은 다음과 같다. 기본 remote-name은 origin.
$ git fetch origin
